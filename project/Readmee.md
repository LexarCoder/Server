# 🚀 Social Media Backend API

A scalable backend system built with Node.js, Express, and MongoDB that supports authentication, content sharing (reels & posts), and social features like followers and feed.

---

## 📌 Features

### 🔐 Authentication
- User Registration
- User Login (JWT आधारित)
- Secure Logout
- Password hashing using bcrypt

### 🎬 Posts / Reels
- Create Reels (video/link based)
- Create Posts
- Save / Bookmark Posts
- View Feed (based on following system)
- Post Collections

### 👥 User System
- Follow / Unfollow Users
- Followers & Following List
- User Profile

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB + Mongoose  
- **Authentication:** JWT (JSON Web Token)  
- **Security:** bcrypt, dotenv
